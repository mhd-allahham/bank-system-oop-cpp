# Bank System (OOP C++)

## Beschreibung
Dieses Projekt ist ein konsolenbasiertes Bankensystem, das in C++ unter Verwendung von objektorientierter Programmierung (OOP) entwickelt wurde.

Das System simuliert reale Bankoperationen und bietet eine strukturierte und modulare Architektur durch die Verwendung von Klassen.

## Funktionen
- Kunden hinzufügen, bearbeiten und löschen
- Kunden suchen
- Ein- und Auszahlungen durchführen
- Geld zwischen Konten überweisen
- Gesamtsalden anzeigen
- Währungssystem (Currency Exchange)
- Benutzerverwaltung
- Login-System mit Berechtigungen

## Verwendete OOP-Konzepte
- Kapselung (Encapsulation)
- Abstraktion (Abstraction)
- Trennung von Verantwortlichkeiten
- Modulares Design

## Datenspeicherung
- Kunden: Clients.txt
- Benutzer: Users.txt
- Überweisungen: TransferLog.txt
- Währungen: Currencies.txt

## Technologien
- C++
- Dateiverarbeitung (File Handling)
- Objektorientierte Programmierung (OOP)

## Projektstruktur
Das Projekt ist in mehrere Klassen unterteilt, die jeweils für bestimmte Funktionen verantwortlich sind:
- Kundenverwaltung
- Benutzerverwaltung
- Transaktionen
- Währungsumrechnung
- Benutzeroberflächen (Screens)

## Ausführung
1. Projekt in Visual Studio öffnen
2. Projekt kompilieren (Build)
3. Programm starten
